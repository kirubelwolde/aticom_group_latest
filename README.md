# ATICOM Prototype

## Setup & Run

1. Clone the repository:
   ```sh
   git clone https://github.com/Awash-systems/Aticom-prototype.git
   cd Aticom-prototype
   ```
2. Install dependencies:
   ```sh
   npm install
   ```
3. Start development server:
   ```sh
   npm run dev
   ```
4. Build for production:
   ```sh
   npm run build
   ```
5. Preview production build:
   ```sh
   npm run preview
   ```
